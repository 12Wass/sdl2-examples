SDL2, C++14 and CMake
=====================

Requirements
------------

* sakemake
* Compiler that supports C++17 (recent version of `g++` or `clang++`)
* SDL2

One way of building with C++14, SDL2 and sakemake
-------------------------------------------------

    sakemake

Running
-------

    sakemake run

Cleaning up the binary file and build directory
-----------------------------------------------

    sakemake clean
